
# Fleet Management Dashboard

## Project Snapshot
An interactive Power BI dashboard designed to analyze fleet operations, driver performance, fuel usage, maintenance costs, and revenue trends. This project helps transform raw logistics data into actionable insights for better decision-making.

---

## Problem Statement
Fleet data is often scattered across multiple systems, making it difficult to track performance, control costs, and understand business impact.

This project solves that problem by building a centralized dashboard that provides clear visibility into both operational and financial metrics.

---

## Project Objective
The goal of this dashboard is to:

- Track driver and vehicle performance  
- Monitor fuel consumption and efficiency  
- Analyze maintenance and repair costs  
- Evaluate revenue and delivery trends  
- Understand customer behavior  
- Identify high-performing regions  

---

## Tools & Technologies Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query  
- Data Modeling  
- Interactive Filters & Slicers  

---

## Dashboard Overview

### Driver Dashboard

- Total kilometers traveled  
- Fuel consumption tracking  
- Maintenance and repair costs  
- Total number of drives  
- Filters by truck type, trailer type, and time  

---

### Driver Tooltip (Detailed View)

- Fuel consumption per KM  
- Fuel cost per KM  
- Average fuel efficiency  
- Average speed  
- Total trips  

---

### Revenue Insights

- Total revenue and delivered orders  
- Goods value, weight, and cubic volume  
- Year-over-year growth  
- Repeat vs one-time customers  
- Revenue trends over time  
- Revenue by city and state
